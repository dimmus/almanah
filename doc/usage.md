## Using Almanah

- **Launch**:

  - From the build tree: `meson devenv -C build -- almanah`
  - From the desktop: run the `Almanah Diary` application.

- **Basic workflow**:

  - Select a date using the calendar button in the header.
  - Type or edit the diary entry in the main text area.
  - Mark an entry as important with the star toggle.

- **Tags**:

  - Toggle the tags panel with the tags button in the header.
  - Add a tag by typing its name and pressing Enter.
  - Remove a tag with the × button on the tag chip.

- **Hyperlinks**:

  - Paste a URL (Ctrl+V) to automatically create a clickable link.
  - Or select text and use the “Add/Remove Hyperlink” menu item.

- **Search**:

  - Press Ctrl+F or use the menu to open Search.
  - Enter a query to find entries matching text or tags.
